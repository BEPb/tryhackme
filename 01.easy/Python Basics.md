## Задание 7
В программировании циклы позволяют программам повторяться и выполнять действия несколько раз. Существует два типа циклов forи whileциклы.

### Циклы While

Давайте начнем с рассмотрения того, как мы структурируем цикл while. Мы можем заставить цикл выполняться бесконечно или (аналогично оператору if) определить, сколько раз цикл должен выполняться на основе условия.
```python
i = 1
while i <= 10:
     print(i)
     i = i + 1
```

Этот цикл while будет выполняться 10 раз, выводя значение переменной i каждый раз, когда он итерируется (циклируется)
. Давайте разберем это: 
- Переменная i установлена на 1
- Оператор while указывает, где должно начинаться начало цикла.
- Каждый раз при выполнении цикла он будет начинаться сверху (выводя значение i)
- Затем он переходит к следующей строке цикла, которая увеличивает значение i на 1.
- Затем (поскольку больше нет кода для выполнения программой) она переходит в начало цикла, начиная процесс заново.
- Программа будет продолжать цикл до тех пор, пока значение переменной i не станет больше 10.

### цикл for

Цикл for  используется для итерации последовательности, такой как список. Списки используются для хранения 
нескольких элементов в одной переменной и создаются с помощью квадратных скобок (см. ниже). Давайте изучим на следующем примере: 
```python
websites = ["facebook.com", "google.com", "amazon.com"]
for site in websites:
     print(site)
```

Этот цикл for, показанный в блоке кода выше, будет запущен 3 раза, выводя каждый веб-сайт в списке. Давайте разберем это:

Список переменных с именем websites хранит 3 элемента
Цикл проходит по каждому элементу, выводя элемент на печать.
Программа прекращает цикл, когда она прошла все элементы цикла.
Чтобы дать реальный сценарий, вы можете создать программу, которая проверяет, находится ли веб-сайт в сети или есть 
ли товар на складе. Вы бы перебирали список веб-сайтов, добавляли бы функциональность внутри цикла для проверки 
веб-сайта и выводили результаты. В комнате " Python для пентестеров " показано, как использовать Python для 
перечисления целей, создания кейлоггера, сканирования сети и многого другого.   

В Python мы также можем перебирать диапазон чисел, используя функцию range. Ниже приведен пример кода Python, 
который выведет числа от 0 до 4. В программировании 0 часто является начальным числом, поэтому счет до 5 — это от 0 
до 4 (но имеет 5 чисел: 0, 1, 2, 3 и 4)  
```python
for i in range(5):
     print(i)
```

По мере того, как программы начинают становиться больше и сложнее, часть вашего кода будет повторяться, записывая 
один и тот же код для выполнения одних и тех же вычислений, и вот здесь на помощь приходят функции. Функция — это 
блок кода, который можно вызывать в разных местах вашей программы.  

Вы могли бы иметь функцию для выполнения расчета, например, расстояния между двумя точками на карте или вывода

функции может  использоваться несколько раз в программе.

```python
def sayHello(name):

```

- Ключевое def слово указывает на начало функции. За функцией следует , nameкоторый определяет программист (и 
  является параметром функции). В нашем примере это sayHello. 
- После имени функции идет пара скобок, ()которые содержат входные значения, данные, которые мы можем передать в 
  функцию. В нашем примере это имя.
- Двоеточие :отмечает конец заголовка функции.
Обратите внимание на отступы в функции. Подобно операторам if, все, что находится после двоеточий и имеет отступ, считается частью функции.

```python
          return 0.99
```

Если мы вызовем calcCostфункцию и передадим "sweets" в качестве параметра элемента, функция вернет десятичное число 
(float). В коде выше мы берем переменную с именем spent и добавляем стоимость "sweets" через функцию calcCost; когда 
мы вызываем calcCost, она вернет число 3,99.

Чтобы открыть файл, мы используем встроенную функцию open(), а параметр "r" означает "read" и используется, когда мы 
считываем содержимое файла. Переменная имеет метод read() для считывания содержимого файла. Вы также можете 
использовать метод readlines() и перебирать каждую строку в файле; полезно, если у вас есть список, где каждый 
элемент находится на новой строке. В приведенном выше примере файл находится в той же папке, что и скрипт Python; 
если бы он находился в другом месте, вам нужно было бы указать полный путь к файлу.    

Вы также можете создавать и записывать файлы. Если вы пишете в существующий файл, вы сначала открываете файл и 
используете "a" в функции open после вызова имени файла (что означает append). Если вы пишете в новый файл, вы 
используете "w" (write) вместо "a". Для ясности смотрите примеры ниже:  
```python
f = open("demofile1.txt", "a") # Append to an existing file
f.write("The file will include more text..")
f.close()

f = open("demofile2.txt", "w") # Creating and writing to a new file
f.write("demofile2 file created, with this content in!")
f.close()
```

Обратите внимание, что мы используем метод close() после записи в файл; он закрывает файл, поэтому дальнейшая запись 
в файл (внутри программы) невозможна.
